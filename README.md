## About me

I'm a **Data Engineer & Data Analyst** based in **Moscow**, focused on designing and operating production-grade data platforms.

I enjoy working across the full data lifecycle — from ingestion and orchestration to modeling, storage, and delivery — with an emphasis on **reliability**, **observability**, and **clear semantics** for downstream analytics.

```text
  Sources ──► Ingestion ──► Orchestration ──► Warehouse ──► Analytics
     │            │              │               │              │
   APIs/DBs    Python/Airflow   DAGs & SLAs   PG / CH / Oracle   BI & DA
```

---

## What I work on

| Area | Focus |
|------|--------|
| **Pipelines** | Batch ETL/ELT, scheduled workflows, idempotent loads |
| **Orchestration** | Apache Airflow — DAG design, retries, monitoring |
| **Storage** | PostgreSQL, ClickHouse, Oracle — schemas, performance, migrations |
| **Engineering** | Python for data jobs, Docker for reproducible environments |
| **Analytics** | Exploratory analysis, metrics, ad-hoc reporting (DA) |
